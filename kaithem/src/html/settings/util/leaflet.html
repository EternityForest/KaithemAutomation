Leaflet maps demo

map tile server Maps

This is just a tester. To actually enable serving maps, load a Map Server module from the Modules Library.

To update or get new maps, either browse around using Marble, or rsync from /home/USERNAME/.local/share/marble/maps/ on another computer.

Use /static/js/leaflet/leaflet.min.js and /static/js/leaflet/leaflet.css to make your own map-enabled modules. Kaithem's builtin tile server is on: /maptiles/tile/{z}/{x}/{y}.png when enabled.
