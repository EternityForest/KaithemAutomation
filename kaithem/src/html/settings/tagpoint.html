Loggers

These loggers control the logging of tag data to history.db

Interval is the maximum rate to log at. No logs are created if there are no changes. Tags that change more often than the interval will always 'flush' the most recent value after sufficient time.

Accumulate determines what happens to multiple readings within an interval. To store the latest, use latest. To store the mean for the whole interval, use average./

Set the interval to 0 to delete the logger. Set history length to 0 to disable deleting old data.
