This device manages one SG1 gateway directly connected via USB port. Gateways interface the server with the SG1 RF protocol, typically at 915MHz with RFM69 modules, but any other RF interface like ESP-NOW could be used.

If you are only using one gatway on the local server, you do not need to edit anything except the serial port. A real MQTT server is not local, as kaithem can use an internal "virtual" server(Names beginning with __virtual__).

Serial Port
Gateway ID(Determines MQTT topic names)
MQTT Server
MQTT Port
RF Profile(7=250k default, lower for longer range and less speed)
Channel Number(Determines the RF frequency)
