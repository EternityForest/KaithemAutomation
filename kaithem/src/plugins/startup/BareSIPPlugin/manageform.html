This is a local/serverless SIP softphone suitable for directly calling at the given address. Use kaithem.devices['Name'].call("number") to place an outgoing call.
dev.hang() is used to hang up any active calls.
Creating multiple is not likely to work at all, as baresip always uses the same JACK name, although it may work if only one at a time is ever active.
