Info

This device just provides web configurability for an MQTT connection, acessible as device.connection, which is the same as a kaithem.mqtt.Connection.

Note that if you delete and recreate this, all subscribers will continue to work the same so long as you keep the same messagebus name. The system tracks subscriptions to busNames independant of any particular connection object, until subscribers are deleted or unsubscribed.

If you use an empty Local Name, one is generated based on the server:port, but subscribers will not automatically carry over if you change the server or port.
